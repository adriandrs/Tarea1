MooTools

Este ejemplo usa MooTools para animar un cuadro cuando se hace clic en él.

                



    
    
    
        
        window.addEvent("domready", function () {
            $("cuadro").addEvent("click", function () {
                this.set("tween", { duration: "500" }).tween("width", "300px");
            });
        });

MooTools se utiliza para animar el cambio de tamaño de un cuadro al hacer clic en él, proporcionando una transición suave.
