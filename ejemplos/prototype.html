Prototype

Este ejemplo utiliza Prototype para actualizar el contenido de un párrafo con texto dinámico.

                



    
        document.observe("dom:loaded", function () {
            $("boton").observe("click", function () {
                $("mensaje").update("¡Texto actualizado con Prototype!");
            });
        });

Prototype simplifica la manipulación del DOM, permitiendo actualizar el contenido de un párrafo al hacer clic en un botón.
